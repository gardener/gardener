## gardenadm join

Bootstrap further control plane nodes or worker nodes and join them to the cluster

### Synopsis

Bootstrap further control plane nodes or worker nodes and join them to the cluster

```
gardenadm join [flags]
```

### Examples

```
# Bootstrap a worker node and join it to the cluster
gardenadm join
```
